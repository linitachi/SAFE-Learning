Saturn Controller

Resource

  • Path to a function
    • Use router { }
    • Links a function to an endpoint
  • Path to an object
    • Use controller { }
    • Provides a set of ready-made endpoints for CRUD behaviour over an object.

controller

let resource = controller {
    index indexAction
    show showAction
    add addAction
    edit editAction
    create createAction
    update updateAction
    patch patchAction
    delete deleteAction
    deleteAll deleteAllAction
}
  • index ‚Äî mapped into GET request at / endpoint. Usually used to render a view displaying list of items, or return whole list of items.
  • show ‚Äî mapped into GET request at /:id endpoint. Usually used to render a view displaying details of particular item, or return single item with given id.
  • create ‚Äî mapped into POST request at / endpoint. Used to create and save new item.
  • update ‚Äî mapped into POST and PUT requests at /:id endpoint. Used to update existing item. Usually replaces original item (keeping id), and requires are fields to be filled in the incoming item.
  • delete ‚Äî mapped into DELETE request at /:id endpoint. Used to delete or deactivate existing item.
  • deleteAll ‚Äî mapped into DELETE request at / endpoint. Used to delete or deactivate all items.

HTTP handler can be composed

Code snipts

module Fuber.Routes

open Giraffe
open Saturn


type Profile = { Username: string }

let getProfile next ctx =
    let profile = { Username = "JW" }

    json profile next ctx
// next ctx //return this , pipeline will return getProfile2 result

// never run
let getProfile2 next ctx =
    let profile = { Username = "QQ" }

    json profile next ctx

let setMyHeader = setHttpHeader "myCustomHeader" "abcd"


let headerPipe =
    pipeline {
        plug setMyHeader
        plug getProfile
        plug getProfile2 // never run because getProfile get something, if getProfile return None, pipeline will continue.
    }

let apiRouter = router { get "/profile" headerPipe }

let router = router { forward "/api" apiRouter }


let app =
    application {
        url "http://0.0.0.0:8080/"
        use_router router
        memory_cache
        use_static "content"
        use_gzip
    }

run app
